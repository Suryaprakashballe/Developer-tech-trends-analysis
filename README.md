# 📊 IBM Data Analyst Capstone Project  
## Technology Skills & Trends Analysis (Stack Overflow Survey 2019)

---

## 📌 Problem Statement
The objective of this project is to analyze global developer survey data to identify current and emerging trends in technology skills, including programming languages, databases, platforms, and frameworks.  
The analysis aims to understand which technologies are gaining adoption, which are declining, and how developer demographics influence these trends to support data-driven decision making.

---

## 📂 Dataset
- **Source:** Stack Overflow Developer Survey (2019) – IBM Skills Network  
- **Type:** Structured survey data  
- **Size:** 60,000+ developer responses  
- **Key Features:**
  - Programming languages (current vs future interest)
  - Databases and platforms
  - Web frameworks
  - Developer demographics (age, education, location)
- **Limitations:**
  - Survey-based data may contain self-reporting bias
  - Some missing and inconsistent values

---

## 🛠 Tools & Technologies
- **Python:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  
- **BI Tools:** IBM Cognos Analytics / Google Looker Studio  
- **Environment:** Jupyter Notebook  
- **Version Control:** Git & GitHub  

---

## 🔍 Approach
1. Collected and validated the survey dataset  
2. Cleaned data by handling missing values, duplicates, and inconsistencies  
3. Performed Exploratory Data Analysis (EDA) to identify trends and patterns  
4. Compared current technology usage with future learning preferences  
5. Built interactive dashboards for visual storytelling  
6. Interpreted results to derive insights and implications  

---

## 📊 Key Insights

### Programming Language Trends
- JavaScript is the most widely used programming language  
- Python continues to show strong growth driven by AI and ML adoption  
- Emerging languages such as Rust and Go are steadily gaining popularity  
- Interest in some traditionally dominant languages is declining  

### Database Trends
- PostgreSQL is the fastest-growing and most preferred modern database  
- Legacy databases such as Oracle and MS SQL Server show declining future interest  
- Cloud-native and open-source platforms are increasingly favored  
- Relational databases are evolving by incorporating NoSQL capabilities  

### Developer Demographics
- Majority of respondents are aged between 21 and 43  
- Most developers hold a Bachelor’s or Master’s degree  
- A large proportion of respondents are based in the United States  
- Web development technologies attract strong developer interest  

---

## 💡 Business & Career Recommendations
- Organizations should modernize technology stacks using open-source and cloud-ready tools  
- PostgreSQL should be prioritized for new database implementations and migrations  
- Developers should focus on Python, SQL, analytics, and cloud platforms  
- Educational institutions should align curricula with industry technology trends  
- Continuous upskilling is essential due to rapid changes in technology adoption
