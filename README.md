# 📊 IBM Data Analyst Capstone Project  
### Technology Skills & Trends Analysis

## 📌 Problem Statement
The objective of this project is to analyze global technology workforce data
to identify current and emerging trends in programming languages, databases,
platforms, and developer demographics.  
The insights from this analysis help organizations and learners understand
which skills are in demand and how the technology landscape is evolving.

---

## 📂 Dataset
- **Source:** IBM / Stack Overflow Developer Survey Dataset  
- **Records:** 60,000+ survey responses  
- **Data Type:** Structured survey data  
- **Key Columns:**
  - Programming languages worked with and desired
  - Databases, platforms, and web frameworks
  - Developer demographics (age, gender, education, experience)
- **Limitations:**
  - Survey-based data may include self-reporting bias
  - Some missing or inconsistent values

---

## 🛠 Tools & Technologies
- **Python:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Dashboarding:** IBM Cognos Analytics
- **Data Handling:** Jupyter Notebook
- **Version Control:** Git & GitHub

---

## 🔍 Approach
1. Imported and explored the dataset to understand structure and data types  
2. Cleaned the data by handling missing values and inconsistent entries  
3. Performed Exploratory Data Analysis (EDA) to identify patterns and trends  
4. Analyzed technology usage and future skill preferences  
5. Built interactive dashboards using IBM Cognos Analytics  
6. Interpreted results to generate actionable insights

---

## 📊 Key Insights
- Python, SQL, and JavaScript are among the most widely used programming languages  
- Cloud platforms and relational databases dominate enterprise usage  
- Data-related roles show increasing demand across industries  
- A clear gap exists between currently used technologies and desired future skills  

---

## 💡 Business & Career Recommendations
- Organizations should invest in upskilling employees in high-demand technologies
  such as Python, SQL, and cloud platforms  
- Educational institutions should align curricula with emerging industry trends  
- Aspiring data professionals should focus on analytics, databases, and cloud tools  
- Continuous learning is critical due to rapidly changing technology requirements
